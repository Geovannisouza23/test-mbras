# Exemplos Detalhados dos Algoritmos

## 1. Análise de Sentimento
- Entrada: "O produto é ótimo, recomendo!"
- Saída: Sentimento positivo (score > 0)

## 2. Cálculo de Influência
- Usuário: user_abc
- Fórmula: golden ratio * (followers + engajamento)
- Exemplo: 1.618 * (1500 + 20) = 2462.36

## 3. Trending Topics
- Hashtags mais frequentes e relevantes no período analisado.
- Exemplo: ["#golang", "#backend", "#mbras"]

## 4. Detecção de Anomalias
- Detecta padrões incomuns de mensagens, picos de engajamento ou conteúdo fora do padrão.
- Exemplo: AnomalyDetected: true, AnomalyType: "spike"

## 5. Flags Especiais
- MbrasEmployee: true se user_id contém "mbras"
- SpecialPattern: true se content tem 42 caracteres e "mbras"
- CandidateAwareness: true se content contém "teste técnico mbras"
